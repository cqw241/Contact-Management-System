# C++ 通讯录管理系统 (Contact Management System)

## 📖 项目介绍
这是一个使用 C++ 编写的控制台程序，用于管理联系人信息。适合 C++ 初学者的第一个练习项目。

## 🚀 功能列表
* **添加联系人**：支持记录姓名、性别、年龄、电话、住址。
* **显示联系人**：列表显示所有信息。
* **查找联系人**：按姓名快速查找。
* **修改联系人**：支持单独修改某一项信息。
* **删除/清空**：管理过期的联系人数据。

## 🛠️ 技术栈
* 语言：C++
* 开发环境：Visual Studio 2026

## 💡 遇到的挑战与解决
1. 在使用 `cin` 和 `getline` 混合输入时遇到了缓冲区换行符的问题，通过使用 `cin.clear()`搭配`cin.ignore(numeric_limits<streamsize>::max(), '\n');` 成功解决。
2. switch语句中的case分支，如果要声明局部变量，必须使用{}创建局部作用域。最好所有case语句都使用{}
3. 严格检查数组越界问题，数组越界会导致严重报错
4. 要深入理解循环和跳转逻辑，最好先画出流程图，理清逻辑后再开始编码。
5. 成功实现分文件编写。学习到了不是const的全局变量应该在.h头文件中使用extern再进行声明。如果需要赋值则在.cpp源文件中赋值。（如本项目中的bool in_flag）
6. 使用指针进行地址传递`contact* conptr`），避免内存拷贝，提高效率。
